# Base Converter — Operation Gopher Protocol

### Status: MISSION READY
    As part of the **CodeCrafters Weekend Mission**, this utility provides a high-speed conversion interface for numerical data across different bases. It is a prerequisite for the SENTINEL String Transformer.

---

## 🛠 Features

The Base Converter handles three primary numbering systems:
* **Hexadecimal (Base 16)**: Standard for memory addresses and encoded data.
* **Binary (Base 2)**: The fundamental language of the SENTINEL system.
* **Decimal (Base 10)**: Human-readable integer format.
